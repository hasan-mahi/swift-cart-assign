1) What is the difference between null and undefined?
Answer: Undefined means value has not assigned and javascript assign it for uninitialized value and null is for Intentionally or manually assigned by the programmer.

2) What is the use of the map() function in JavaScript? How is it different from forEach()?
Answer: map() function is used to create a new array with updated elements and does not impact the original array whereas forEach() is loop through the array does not create new array and mostly used for loop through array element

3) What is the difference between == and ===?
Answer: (==) is only check the value not the reference or type but (===) is check the value and reference or type of that value;

4) What is the significance of async/await in fetching API data?
Answer: It allows us to write asynchronous code that behaves like synchronous code which simplifies error handling, and makes the code flow more perfectly.

5) Explain the concept of Scope in JavaScript (Global, Function, Block).
Answer: Global: Variables that declared outside of any function or block called a global scope.They are accessible from anywhere in the JavaScript program.

Function: Variables declared within a function are called the function scope. They are only accessible within that function where they are defined and any nested functions.

Block: Variables declared with let or const inside a block of code enclosed by curly braces {}.They are only accessible within that block where they are defined.
